# CryptoBuddy 🌟
**Your First AI-Powered Financial Sidekick!**

This simple rule-based chatbot gives beginner-friendly advice on cryptocurrency investment based on profitability and sustainability.

---

## 🔧 Features
- Friendly conversational bot
- Suggests coins based on:
  - 🌱 Sustainability (low energy use + high score)
  - 📈 Profitability (rising price trend + market cap)
- Pure Python logic (no advanced AI required)

---

## 📦 How to Use
1. Run `crypto_chatbot.py` in your IDE or Jupyter Notebook.
2. Ask questions like:
   - “What’s the most sustainable coin?”
   - “Which crypto is trending up?”
3. Type `exit` to quit.
